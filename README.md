## ðŸš€ Getting Started

### Prerequisites

Pastikan Anda sudah menginstall:
- [Node.js](https://nodejs.org/) (v14 atau lebih tinggi)
- npm atau yarn

### Installation

1. **Clone repository**
```bash
   git clone https://github.com/username/dabang-dashboard.git
   cd dabang-dashboard
```

2. **Install dependencies**
```bash
   npm install
```

3. **Run development server**
```bash
   npm run dev
```

4. **Open browser**
```
   http://localhost:5173
```
